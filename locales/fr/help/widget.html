Formatage du widget

Affichage du bloc :
%s sera remplacé par le bloc de code
Affichage d'un élément :
  • %1$s
    • sera remplacé par l'URL de téléchargement du fichier

  • %2$s
    • sera remplacé par le titre du fichier (ou le nom du fichier si il n'a pas de titre)

  • %3$s
    • sera remplacé par nom du fichier

  • %4$s
    • sera remplacé par l'icône vers la prévisualisation
