Formatage du widget

Affichage du bloc :
%s sera remplacé par le bloc de code
Affichage d'un élément :
  • %1$s sera remplacé par l'URL de téléchargement du fichier
  • %2$s sera remplacé par le titre du fichier (ou le nom du fichier si il n'a pas de titre)
  • %3$s sera remplacé par nom du fichier
  • %4$s sera remplacé par l'icône vers la prévisualisation

Par défaut, le champ contient le code suivant :
<li><a href="%1$s" title="%3$s">%2$s</a> %4$s</li>

Les fichiers n'apparaissent pas dans le plugin

Rendez-vous dans la médiathèque et naviguez dans tous les dossiers pour que Dotclear prenne en compte les fichiers.
Videz le cache du navigateur si les fichiers n'apparaissent toujours pas sur la page publique du Gestionnaire de téléchargements.
