Widget formatting

Block display:
%s will be replaced by the code block
Item display:
  • %1$s will be replaced by the file's download URL
  • %2$s will be replaced by the file title (or the file name if there is no title)
  • %3$s will be replaced by the file name
  • %4$s will be replaced by the icon to the preview

The files do not appear in the plugin

Visit the library and browse through all records for Dotclear take into account files.
Empty the browser cache if the files still do not appear on the public page of the Download Manager.
